3D Survival Prototype Introduction – 24/03/2026

Currently, I am developing a 3D survival game prototype in Unity. Using the experience gained from my previous 2D concept game Feathers of Ember, I aim to create a functional survival game concept in 3D.

Taking a prototype-first approach allows me to focus on the core mechanics and programming aspects of a 3D survival game, rather than spending significant time on detailed world design and assets. Once the game reaches a playable state with a satisfying gameplay loop, I may continue developing the prototype further or start a new project based on the lessons learned.

My goal with this website is to move away from long-form blog posts and instead adopt a more structured, short-form, bullet-point style format below. This is intended to make the site more portfolio-focused and easier to understand at a glance.

Completed:

  • Basic gameplay test environment
  • Simple 3D assets including a tree, axe, campfire, and mushroom
  • Animated campfire flames using a particle system
  • Utilised a third-party first-person controller
  • Animated axe swing and tree falling
  • Implemented tree resource breakdown (tree → log → stick) on axe hit
  • Enabled stick collection, carrying, and dropping
  • Created an accessible inventory UI

Up next:

  • Follow tutorial to learn how to develop an inventory system
  • Make mushrooms a forageable item
  • Implement an interactable campfire that requires wood to light
  • Create a ferro rod and striker for fire starting
